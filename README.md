# 🌊 Hydrobiology App

A modern **Hydrobiology Web Application** built with React that allows users to explore aquatic species using real-world biodiversity data.

This app combines data from:
- iNaturalist (for species images)
- GBIF (for scientific taxonomy)

---

## 🔬 Features

- 🐟 Explore aquatic species with real images  
- 🧬 Scientific classification (Genus, Family, Order)  
- 📊 Research-focused UI for scientists & learners  
- 🔄 Pagination with "View More" functionality  
- ⚡ Fast and responsive design using Tailwind CSS  
- 🌊 Clean hydro-themed UI  

---

## 🚀 Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **APIs Used:**
  - iNaturalist API (images & species data)
  - GBIF API (taxonomy & scientific classification)

---

## 🧪 How It Works

1. Fetch species data from iNaturalist API  
2. Extract scientific names  
3. Fetch taxonomy data from GBIF API  
4. Merge both datasets  
5. Display enriched species cards
